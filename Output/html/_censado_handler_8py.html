Detailed Description

CensadoHandler is the interface to DB this particular file only takes care of the sensor measure DB part interface.

In order to be able to change of DB engine in the future if it's needed, this interface was implemented. So all the system uses these functions to acces DB instead of direct interaction with entity, which let to change the DB engine and all the changes will be need only within the functions of this module
