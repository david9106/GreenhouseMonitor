get JSON object/list from url

Creates an AJAX asyncronous GET request to a given url asking for all the sensor measures in a given period of time and waits for a JSON object/list as response

Parameters
[in]urlThe server URL, it must has the prefix "http://"
[in]successHandlerThe function name that will be triggered in case that status 200 is responded by server
[in]errorHandlerThe function name that will be triggered in case that an error status is responded by server
[out]sensor_typeThe measure's sensor type required
[out]date1The start of the date range, must be string date format
[out]date2The end of the date range, must be string date format
