Detailed Description

Retrieves data form DB and gives it to user in JSON format.

Uses the same command format of the ConfigProvider class. A json request with a "Tipo" member that indicates the command and the command arbitrary named, but using CamelCase practice No parameters It inherits from webapp2.RequestHandler
