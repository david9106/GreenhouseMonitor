Detailed Description

Receive the sensor JSON objects from gateway, parses them, call the comparator module and DB module.

This Function receive a JSON object and parse ir, then verify if the sensed data type excist in datastore, if not creates a new one entity in datastore if not updates the excisting one, then compares the receive data with the excisting limits in datastore
